配置 github

创建代码库，在Repository name下填写yourname.github.io

（yourname 一定要是你的用户名）

代码库创建成功后点击Settings，找到Github pages 点击Luanch automatic page generator，这样Github将会自动替你创建出一个gh-pages的页面，几分钟后yourname.github.io 就可以登陆了。
